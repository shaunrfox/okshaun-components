  ### Font Variants

  The `font-variation-settings` property in CSS allows you to fine-tune attributes like weight, width, slant, or custom features on a variable font. Our main body font ([Recursive](https://www.recursive.design/)) supports the following variations:

  - `MONO` (monospace): 0 to 1
  - `CASL` (casual): 0 to 1
  - `wght` (weight): 300 to 1000 (we're using `fontWeight` to control weight instead of this)
  - `slnt` (slant): -15 to 0
  - `CRSV` (cursive): 0, 0.5, 1

  These have been built into shorthand properties for convenience, and can be applied with the `fontVariant` prop:

  ```jsx
    <Text fontVariant="body-italic">...</Text>
  ```
