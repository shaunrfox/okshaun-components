### Token Types

  <Grid columns={{ base: 1, '@/2xl': 2 }} gap="40" alignItems="flex-start">
    <Flex flexDirection="column" gap="4">
      <Text textStyle="body.lg" fontWeight="bold" color="text">Primitive Tokens</Text>

      Raw design values defined in `src/styles/primitives/`. These are the foundation and don't change based on theme.
    </Flex>

    <Flex flexDirection="column" gap="4">
      <Text textStyle="body.lg" fontWeight="bold" color="text">Semantic Tokens</Text>

      Theme-aware tokens defined in `src/styles/semantics/`. These reference primitive tokens and automatically switch values for light/dark themes.
    </Flex>
  </Grid>

### Best Practices

  <Grid gridTemplateColumns="auto auto 1fr" gap="12" alignItems="flex-start">
    <Icon name="check" />
    <Text fontWeight="bold" color="text">Use semantic tokens</Text>
    <Text>Prefer `bg.surface` over raw color values for theme compatibility</Text>
    <Icon name="check" />
    <Text fontWeight="bold" color="text">Follow spacing scale</Text>
    <Text>Use predefined spacing values (4, 8, 16, 24, etc.) instead of arbitrary values</Text>
    <Icon name="check" />
    <Text fontWeight="bold" color="text">Use text styles</Text>
    <Text>Choose appropriate `textStyle.*` tokens for consistent typography hierarchy</Text>
    <Icon name="check" />
    <Text fontWeight="bold" color="text">Test in both themes</Text>
    <Text>Semantic tokens automatically adapt, but verify the result looks good in light and dark modes</Text>
  </Grid>
